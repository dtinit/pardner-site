What is this? How does it work?

This is a demo of what a data donation website looks like out of the box when you download the Pardner site.

You may customize this site for your own donation purposes, or extend the library created by the Data Trust Registry for use for your own data donation solicitation needs. The Data Trust Registry library currently supports donations of data from Tumblr.

If you have any questions, contact the Data Trust Registry.
